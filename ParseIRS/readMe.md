# ParseIRS 使用方法

1、首先将doc文件手动转换位docx文件，由于文本只能处理docx文件

2、将docx文件路径赋值给脚本文件中的docName 变量，相对路径或绝对路径均可

3、电脑上安装python2.7

4、运行 python ParseIRS.py  irs_name，等几秒即可输出结果，例如：

```
python ParseIRS.py IRS_CHX002_D18F0_xHCICtrl_R091.docx

或者

python ParseIRS.py D:\CHX001_Tool\IRS_CHX002_D18F0_xHCICtrl_R091.docx
```
